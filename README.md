# RPG-VideoGame

Proyecto de combate RPG en texto hecho con PHP basico.

Para ejecutar el programa:

```bash
php index.php
```

Archivos principales:

- `Origen.php`: clase abstracta con los datos y metodos comunes.
- `Personaje.php`: clase hija para el jugador principal.
- `Enemigo.php`: clase hija para los enemigos.
- `Habilidad.php`: clase que representa una habilidad.
- `CombatienteInterface.php` y `HabilidadInterface.php`: interfaces del proyecto.
- `Ronda_Ataque.php`: clase que maneja el ataque aleatorio y el golpe critico.
- `index.php`: simulacion completa, sin entrada del usuario.
- `DOCUMENTACION.md`: explicacion detallada para estudiar y defender la tarea.
